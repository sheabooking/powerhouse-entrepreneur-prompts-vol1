## What This Is

This is a full guided coaching experience built for Claude. It walks entrepreneurs through a structured, 5-phase self-discovery journey — from finding their values and purpose, to building a real schedule, validating a business idea, and breaking through the mindset blocks that have kept them stuck.

By the end, Claude delivers **3 personalized coaching documents** and **2 working templates** — built from everything the person shared throughout the journey.

**This is not a chatbot. This is a coach.**

---

## The 5 Phases

| Phase | Focus |
|-------|-------|
| Phase 1: The Clarity | Values + Purpose |
| Phase 2: The Vision | Goal-setting + Obstacles |
| Phase 3: The Structure | Time management + Scheduling |
| Phase 4: The Build | Business validation + Offers |
| Phase 5: The Mindset | Self-doubt + Mental blocks |

---

## What They Walk Away With

1. **Full Picture Analysis** — patterns, contradictions, and insights drawn from their own words across the entire journey
2. **Goals on an Eisenhower Matrix** — every goal sorted visually by what needs attention now vs. what can wait, with baby steps for each
3. **One Powerful Question** — a single question designed specifically for them to sit with after the journey
4. **Weekly Planning Template** — reusable, editable, designed for real life
5. **Goal Milestone Template** — tracks baby steps, milestones, and progress for each major goal

## Install This Skill

```bash
claude plugin install https://github.com/sheabooking/powerhouse-entrepreneur-prompts-vol1
```

Or download the `.skill` file and install manually:
```bash
claude plugin install /path/to/powerhouse-business-foundations.skill
```

Once installed, just start a conversation. Say anything — "I'm ready," "let's go," "I need clarity" — and the coaching journey begins.
